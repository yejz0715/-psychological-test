# MBTI 유형별 심리테스트 웹사이트 


<h3>프로젝트 소개</h3>

* UI 부터 배포까지 웹사이트의 전체적인 흐름을 파악할 수 있었던 프로젝트 입니다.

<h3>사용 기술</h3>

* html, javaScript, css
